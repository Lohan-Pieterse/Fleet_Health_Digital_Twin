# AI Usage Disclosure

I used AI tools during this project to optimize my workflow, structure my tasks, and assist with debugging. Here is a breakdown of how they were utilized:

### 1. GitHub Copilot
* Used for code autocomplete to increase development speed and handle boilerplate code efficiently.

### 2. Claude (Task Organization & Documentation)
* **Task Breakdown:** I provided the project requirements to Claude with the prompt: 
  > "View the task below and give me a detailed list of steps I should complete. Give me only the steps and no answers. I want to use this as a checkbox to complete the project."
* **Documentation Template:** To ensure a clean layout for my final documentation, I used the prompt:
  > "Please write a basic readme.md file that I can use to create mine for a project. Please use the style that is used for a normal readme.md"

### 3. Debugging & Problem Solving
* When encountering configuration issues with `docker-compose`, I shared the file and error logs with AI. Rather than asking for a direct copy-paste solution, I prompted it to help me identify the root cause and guide me through the troubleshooting steps so I can resolve similar issues independently in the future.
